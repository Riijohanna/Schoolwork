Transitions

Description
Make a circle which gets bigger and changes color while you hover. Turn it around also.
Solution
CSS Example 
      .circle {
        background: radial-gradient(circle at 100px 100px, #de5cff, #000);
        -webkit-transition: width 1s, height 1s, -webkit-transform 3s;
        transition: width 1s, height 1s, transform 3s;
      }
      .circle:hover {
        -webkit-transform: rotate(180deg); /* Safari */
        transform: rotate(180deg);
        background: radial-gradient(circle at 100px 100px, rgb(205, 200, 66), #000);
      }
