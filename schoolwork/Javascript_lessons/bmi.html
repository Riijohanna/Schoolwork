How much is your BMI?

Description
Make a program that calculates body mass index. The BMI is defined as the body mass divided by the square of the body height, and is universally expressed in units of kg/m2, resulting from mass in kilograms and height in metres.
Solution
Body mass (kg)
Your height (m)

Code Example 
  function myFunction() {
    var mass = Number(document.forms['ask']['mass'].value);
    var height = Number(document.forms['ask']['height'].value);
    var total = mass/(height*height);
    document.getElementById('demo').innerHTML = 'Your body mass is '+total+'kg/m.';
  }
