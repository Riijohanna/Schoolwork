Tell me 20 numbers

Description
Make a program that asks 20 numbers from user. After that the program prints out how many of those numbers where even.
Solution
Tell me the numbers (prompt)

Code Example 
    function myFunction() {
      var evens = 0;
      for (var i = 0; i < 20; i++) {
        var number = Number(prompt('Tell me the numbers'));
        if (number % 2 == 0) {
          evens++; //evens=evens+1;
        }
      }
      alert('Number of evens is ' +evens);
    }
